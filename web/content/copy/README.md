## What this folder is

This folder contains the copy extracted from your current site pages, organised into template-friendly Markdown documents (one per page, plus shared global copy).

## Files

- `00-site-wide.shared.md`: Shared items (navigation labels, footer/legal, global disclaimer, primary CTAs).
- `01-home.md`: Home page copy.
- `02-for-you.md`: For You? page copy.
- `03-how-it-works.md`: How It Works page copy.
- `04-privacy.md`: Privacy Notice page copy.
- `05-about.md`: About page copy.
- `06-contact.md`: Contact page copy.

## Suggested mapping to a new template

Typical components/slots to map into:

- Hero: H1 + subhead + primary CTA
- Feature blocks: “What I do”, “Why it’s worth doing”
- Stepper: How-it-works steps
- Trust & privacy: privacy summary + processors + cookie statement
- CTA blocks: “Get in touch” / “Get started”
